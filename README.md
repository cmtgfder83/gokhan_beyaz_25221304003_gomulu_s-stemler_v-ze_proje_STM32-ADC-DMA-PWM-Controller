# STM32 ADC-DMA-PWM Kontrol Sistemi 🚀
**Öğrenci:** Gökhan Beyaz  
**No:** 25221304003

## 📝 Proje Tanıtımı
Bu proje, **STM32F030F4Px** mikrodenetleyicisi üzerinde ADC, DMA ve PWM birimlerinin entegre çalışmasını sergiler. PA0 pinindeki potansiyometre ile PA9 pinindeki LED'in parlaklığı kontrol edilmektedir.

## 🛠 Teknik Konfigürasyon
| Birim | Pin | Görev |
| :--- | :--- | :--- |
| **ADC** | PA0 | Potansiyometre Okuma |
| **PWM** | PA9 | LED Parlaklık Kontrolü |
| **EXTI** | PA1 | Sistem On/Off Butonu |
